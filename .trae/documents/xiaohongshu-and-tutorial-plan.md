# 小红书分享文案与教程集成计划

## 1. 网页内嵌教程实现评估 (可行性：极高 / 难度：极低)
将教程直接整合进网页内部**非常简单**，不仅能提升产品完整度，还能给通过小红书引流来的新用户提供无缝的 Onboarding 体验。

**实现方案：**
*   **组件入口**：在 `HeroSection.tsx` 的 "Start Practicing" 按钮旁边新增一个次级按钮 `How it Works` (带一个播放或书本图标)。
*   **UI 呈现**：点击后弹出一个居中的 Modal / Dialog。
*   **内容结构**：弹窗内可以采用简单的图文或分步结构介绍产品的四大核心能力：
    1.  **配置**：去 Settings 填入 API Key 即可免费使用。
    2.  **沉浸写作**：打字停顿时出现灰色预测文本 (按 Tab 采纳)；句末自动拼写语法检查 (红波浪线 + 绿字替换)。
    3.  **划词润色/翻译**：选中中文直接给出英文地道表达；选中英文自动进行高级词汇替换或句型重写。
    4.  **双考官打分**：交卷后停止计时，获取 Native 老师 (Kelly) 与结构老师 (Jieming) 的双视角点评，一键翻译。
